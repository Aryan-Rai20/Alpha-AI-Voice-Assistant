# Alpha AI Voice Assistant 🎙️🤖

## About the Project

Alpha is a Python-based AI Voice Assistant developed as a part of my Python Development Internship (Task 3). The main aim of this project is to perform daily computer tasks using voice commands and provide intelligent responses with the help of Google Gemini AI.

The assistant can understand voice commands, reply using speech, perform system operations, search the web, provide live weather updates, and answer general questions using AI.

---

## Features

- 🎤 Voice Command Recognition
- 🔊 Text-to-Speech Response
- 🤖 Google Gemini AI Integration
- 🌦️ Live Weather Information
- 🔋 Battery Percentage
- 📸 Take Screenshot
- 🌐 Open Google
- 🔍 Google Search
- ▶️ Open YouTube
- 🎥 Search on YouTube
- 📝 Open Notepad
- 🧮 Open Calculator
- 🕒 Tell Current Time
- 📅 Tell Current Date
- ❌ Exit Assistant

## Installation

Clone the repository

```bash
git clone <repository-link>
```

Go to project folder

```bash
cd Voice Assistant
```

Install all required libraries

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
